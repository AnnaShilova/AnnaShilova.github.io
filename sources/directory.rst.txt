**Вкладка "справочники"**
=========================

--------------------------------------------------------------

Вкладка разделы содержит кнопки на остальные таблицы требуемые для проекта, а именно:

* Таблица "Типы машин"
* Таблица "Свободные номера машин"
* Таблица "Техосмотр"

Как открыть таблицу
-------------------

Для того что бы открыть нужную таблицу наведите курсор на вкладку "справочники", выберите нужную таблицу и нажмите на нее. В данном окне будет обтображаться информация по выбранному разделу. Вврху окна доступны три кнопки, работа которых будет описана далее. Работа кнопок будет приведена на примере таблицы "тип машин". Для остальных таблиц действие кнопок аналогичное.

Добавление записи
~~~~~~~~~~~~~~~~~
 #. нажмите на кнопку добавить, что бы добавить новый тип машины;
 #. в открывшемся окне заполните необходимую информацию для добавления нового типа машины;
 #. после того, как нужная информация будет введена, нажмите кнопку "ОК";
 #. окно закроется и информация будет добавлена.

Изменение записи
~~~~~~~~~~~~~~~~
 #. нажмите на тот тип машины информацию которого хотите изменить;
 #. после нажмите кнопку "Изменить";
 #. в окне измените данные тех параметров, которые Вам нужны;
 #. нажмите кнопку "ОК" и Ваши изменения будут сохранены.

Удаление записи
~~~~~~~~~~~~~~~
 #. нажмите на тот тип машины информацию которого Вы хотите удалить;
 #. после нажмите кнопку "Удалить".


Работа данных кнопок анологична для остальных справочников.
Для того, чтобы выйти из программы нажмите кнопку "Выход".
